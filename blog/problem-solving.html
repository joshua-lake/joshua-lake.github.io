Problem Solving


Got blocked on a simple problem :

While doing the FizzBuzz challenge, I fell pretty short at the first hurdle.. How to identify all multiples of a number. The first thing I did was jump onto Google, but didn't have much luck there. So I reached out to my facilitator who was able to set me straight and help me fully understand the remainder operator. It was pretty frustrating stumbling at the first hurdle but having someone explain a concept one on one is a great way to solidfy the knowledge of how something might work.


Solved a problem in an elegant way :

Not a code related problem.. When I was working in VFX, we had some footage come in that was in the wrong resolution. We were having issues converting it without affecting the colourspace. After asking some peers for help and some googling, I was able to get the conversion done. I actually enjoyed solving this problem because I felt like I had the tools to do so, and I learnt a new technique to convert footage!


Problem solving techniques

Psuedocode

As someone who is new to programming, this technique is completely new to me. However, over the last 2 sprints I've certainly come to realise how important using psuedocode can be. Being able to articulate the code in an everyday language helps identify exactly what you are trying to achieve, or what's going wrong and how you're going to go about fixing it.


Trying something

This is my favourite, I like getting stuck in! Although I need to be a bit careful as sometimes a more restrained aproach would be more logical.. Psuedocode for example!


Rubber ducky method

Another method I quite like. Externalising the problem can be very helpful in some situations. I've drawn up my own rubber ducky on my whiteboard.. 'Perry the Panda'!


Reading error messages

Confident in reading where the error is, but some of the messages are still a little unclear to me.


Console.logging

Another new one for me, and not overly confident with this one. I understand how it works, but at times uncertain where in the code I should be putting the console.log or which part of a function I should be console.logging out.


Googling

Pretty familiar and confident with ol' Dr Google.


Asking your peers for help

When appropriate, this is a great way to share knowledge. Teaching something is the best way to reitierate it for yourself so people in learning spaces tend to be very happy to help out


Asking coaches for help

Just today i spent some time on a call with my foundations facilitator. They were very helpful and talked me through the issues I was facing and helped me understand some concepts I was struggling with.


Improving your process with reflection

I like this. Reflecting provides a way to be honest with yourself and acknowledge the mistakes you might have made and learn from them.
